Brick Destroyer ...in JavaScript

Start each level by CLICKING with the MOUSE CURSOR inside the game area.
Move the slider left and right by moving the MOUSE CURSOR over the game area.
Hold the SHIFT key to make the slider more bouncy.
Press the ESCAPE key to pause/resume the game.
Reload the page to start from the beginning.
Created by Momchil Atanasov. Game originally developed in Java.
Source code available at https://github.com/momchil-atanasov/brick-destroyer.
